TABLES

Create a simple table

Example : 

		
		<table>
			<tr>
				<th> Heading 1 / Column 1 </th>
				<th> Heading 2 / Column 2 </th>
			</tr>
			
			<tr>
				<td> Row 1 data Column 1 </td>
				<td> Row 2 data Column 2 </td>
			</tr>
			
			<tr>
				<td> Row 2 data Column 1 </td>
				<td> Row 2 data Column 2 </td>
			</tr>
		</table>
			
		
		for adding border
			<style>
			table,th,td
			{
			   border:4px solid black;
			}
			</style>

Output :

Heading 1/ Column 1 Heading 2/Column 2
Row 1 data column 1 Row 1 data column 2
Row 2 data Column 1 Row 2 DATA Column 2

Spaning columns or rows

	
		<table>
			<tr>
				<th> Name </th>
				<th colspan="2"> class</th>
			</tr>
			
			<tr>
				<td> Chirag </td>
				<td>MCA </td>
				<td> 3rd sem </td>
			</tr>
		</table>

Output :

Name Class
Chirag MCA 3rd sem

rowspan is also work same as colspan , rowspan work in rows

In the table more property exist like: thead , tbody , tfoot , caption
