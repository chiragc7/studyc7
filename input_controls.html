Input Controls

Text

<input type="text">

The default width of a text field inputs is 20 characters. This can be change by specifiying a value for the size.

<input type="text" size ="50">

if you need a multi-line text input for subtantial amount of text , use a <textarea> element

Checkbox and Radio button

<input type ="checkbox">
<input type ="radio">
