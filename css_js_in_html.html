CSS and JS

CSS in HTML

Use the link in the document head

<link rel="stylesheet" type="text/css" href="stylesheet.css" >

Internal stylesheet

example :

		<head>
			<style>
				body{
				color:green;
				}
			</style>
		</head>

Multiple Internal css can be include in a program

Inline style

example :

		<span style = " color:red "> This text is red </span>

Javascript in HTML

Link to add external Js file

example :

<script src ="example.js"> </script>

Directly Include

example :

		<script>
			js code
		</script>
