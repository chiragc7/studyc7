LIST

ORDER LIST

An order list can be create with the <ol> tag

example:

			<ol>
				<li> item </li>
				<li> another item </li>
			</ol>

output:

  1. item
  2. another item

Manually change the number

example:

			<ol start="3">
				<li> item </li>
				<li> another item </li>
			</ol>

output:

  1. item
  2. another item

Change the type of number

type : "1" | "a" | "A" | "i" | "I"

1 ,2 ,3 ,4
a , b , c ,d
A , B , C , D
i , ii , iii , iv
I , II , III , IV

UN-ORDER LIST

An un-order list can be create with the <ul> tag

example: 

			<ul>
				<li> item </li>
				<li> another item </li>
			</ul>

output:

  • item
  • another item

NESTED LIST

			      <ul>
				  <li> item 1 </li>
				  <li> item 2
					<ul>
					   <li> sub item 1 </li>
					   <li> sub item 2 </li>
					</ul>
				   </li>
				   <li> item 3 </li>
				</ul>

output:

  • item 1
  • item 2
    • sub item 1
    • sub item 2
  • item 3
